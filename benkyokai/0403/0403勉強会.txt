
◯管理画面
投稿：時系列順に分類（ブログ、お知らせ等）
	カスタム投稿：複数種類の投稿を作れる
固定ページ：単体のページ（お問い合わせ、会社概要等）、ページごとの親子関係を持てる

外観＞テーマ：テーマを選べる、カスタマイズから編集可
　＞ウィジェット：細かく変更できる
　＞テーマエディター：ｃｓｓをいじれる

設定：元号変更ができる、サイト名も変更可
　＞表示設定：記事一覧数を変更可
　＞パーマリンク設定：urlの設定が変えられる

◯テーマの作成（自作方法）
・Wordpressフォルダ＞wp-content>themas>の中に適当なフォルダを作成
・作成したフォルダ内にindex.php style.cssを作成
・style.cssのコメントからテーマの詳細説明が書ける（参照）
・index.phpにheadを書く（特定のコードの直後にWordpressのコードを挿入）

☆固定ページの作成
固定ページから固定ページを追加できる

☆メニューの作成
・管理画面からメニュー名を入力し、ページを選択して保存する
・下の「メニューの位置」にチェックを付ける
・固定ページの「すべて表示」から作成したページを全て表示・選択・追加できる

◯ホームページの設定
設定＞表示設定　ホームページの表示を固定ページでホームページを選択する
　ホームページや記事を投稿するページを選択できる

◯phpコード
・<?php get_○○(); ?> は決まったものしか読み込めない（header, footer, side)
	それ以外を読み込みたいときは　<?php get_template_part('phpファイル名'); ?> と記述する

<?php if(条件): ?> 処理 <?php endif; ?>	// if
<?php else: ?> 処理 <?php endif; ?>	// else

<?php get_category(); ?>	// 複数のカテゴリを表示
<?php get_category('/'); ?>	// 単体のカテゴリを表示、カテゴリ間に記号を入れたいときに使う
